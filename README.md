## Struktura folderów

```
WDED_SOLO/
├── main.py                
├── analysis_utils.py        
├── data_utils.py             
├── zakupy-online.xlsx        
└── requirements.txt          
```

## Jak uruchomić projekt

1. Zainstaluj zależności:
```bash
pip install -r requirements.txt
```

2. Uruchom główny skrypt:
```bash
python main.py
```

## Wymagania

- Python 3.9+
- Pakiety: `pandas`, `matplotlib`, `seaborn`, `mlxtend`, `openpyxl`, `python-docx`
